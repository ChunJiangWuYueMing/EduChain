# 图片素材清单

请将以下图片放入此目录（`public/images/`）：

## 必需素材

| 文件名 | 说明 | 规格建议 |
|--------|------|---------|
| school-logo.png | 西南交通大学校徽（白色/浅色版本，用于深色导航栏背景） | 80×80px 或更大，PNG 透明底 |

## 可选素材（增强视觉效果）

| 文件名 | 说明 | 规格建议 |
|--------|------|---------|
| bg-buildings.png | 校园建筑轮廓线水印（登录页背景右侧装饰） | 1200×800px，PNG 半透明，浅蓝灰色调，opacity 建议 10%-15% |

> 如果没有 bg-buildings.png，登录页背景仅显示纯色渐变，不影响功能。
>
> 所有图片在 Vue 组件中通过 `/images/xxx.png` 路径引用（Vite 会从 public/ 目录直接提供）。
