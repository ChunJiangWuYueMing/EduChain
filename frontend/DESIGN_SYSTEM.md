# EduChain 前端设计系统

## 1. 项目定位

EduChain 是面向区块链综合实验场景的 B 端管理系统。系统用于展示链上教育数据、课程实验记录、证书存证、交易记录、节点状态等信息。

整体设计应保持清晰、可信、简洁、偏技术感，不做过度花哨的装饰。页面风格以“高校科技平台 + 区块链可信数据”为主。

## 2. 主色规范

系统主色采用西南交通大学校徽蓝：

- RGB：0, 121, 186
- HEX：#0079BA
- CSS 变量：--primary-color

主色主要用于：

- 左侧导航选中态
- 主要按钮
- 页面重点数字
- 顶部系统标识
- 链上可信状态、进度条等强调元素

## 3. 基础颜色

- 主色：#0079BA

- 主色浅色背景：#E6F4FA

## 4. 页面布局

B 端后台采用固定布局：

- 左侧侧边栏
- 顶部状态栏
- 右侧主内容区

推荐尺寸：

- 侧边栏宽度：248px
- 顶部栏高度：64px
- 内容区 padding：24px
- 页面最小宽度：1200px
- 暂不考虑移动端适配

## 5. 素材规范

所有图片素材统一放置在：

src/assets/images/

当前固定素材：

- swjtu-logo.png：西南交通大学校徽，不允许手绘替代

## 6. 组件风格

卡片：

- 背景：#FFFFFF
- 圆角：16px
- 边框：1px solid #E5E7EB
- 阴影：0 10px 30px rgba(15, 23, 42, 0.06)

按钮：

- 主按钮使用 #0079BA
- hover 使用 #00649A
- 圆角：10px
- 高度：36px / 40px

表格：

- 表头背景：#F9FAFB
- 行 hover 背景：#F8FAFC
- 状态字段使用胶囊标签

交互：

- 导航选中态使用主色背景
- hover 使用浅蓝背景
- transition 控制在 0.2s 左右

## 7. 登录页设计规范

登录页采用独立大屏布局，不使用后台管理页的侧边栏与顶部栏。

### 布局

- 页面宽度：100vw
- 页面高度：100vh
- 左侧品牌栏宽度：260px
- 右侧内容区占满剩余空间
- 登录卡片位于内容区偏左位置
- 产品说明区位于登录卡片右侧

### 色彩

登录页继续使用西南交通大学校徽蓝作为主色：

- 主色：#0079BA
- 登录按钮深蓝渐变：#005BBB 至 #004AA8
- 左侧品牌栏深蓝：#003A70 / #002D5C
- 页面背景：#F7FBFF
- 错误提示背景：#FFF1F1
- 错误提示边框：#F3B7B7
- 成功状态点：#16C35B
