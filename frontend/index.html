发布资料

发布学习资料

选择资料文件，填写课程信息，设置下载价格和访问权限，提交后完成链上存证。

上传资料获得积分 资料登记成功后发放 +20 EDU 积分。
文件指纹（SHA-256） 同时生成内容相似度（SimHash）用于后续校验。

三步发布

选择文件、填写资料信息、设置价格与权限。

📁 选择资料文件或拖拽到此处

资料文件：支持 PDF、Word、PPT、TXT、Markdown（最大 50MB）

发布后会发生什么

  • 1文件入库保存资料并提取可验证文本。
  • 2生成文件指纹计算文件指纹（SHA-256）与内容相似度（SimHash）。
  • 3写入链上记录完成资料存证并发放 EDU 积分奖励。

更适合课堂演示

发布结果会展示资料 ID、文件指纹、文本长度和定价，方便你切到资料广场或校验页继续演示完整链路。

权限建议

公开下载 适合通识笔记。
仅本课程学生可下载 适合同班共享。
仅指定地址可下载 适合实验材料定向分发。

真伪校验

上传文件与链上记录比对，判断资料是否被篡改。

暂无校验结果请输入资料 ID 并上传待校验文件，系统将比对链上文件指纹。

校验规则

用通俗指标解释链上可信逻辑。

完整性文件指纹是否一致
内容相似度判断文本内容是否明显变化
链上记录来自资料登记合约（MaterialRegistry）
  • 1输入资料 ID定位链上登记的原始指纹。
  • 2上传待校验文件本地重新计算文件指纹。
  • 3输出结论展示匹配、相似度和篡改风险。

审计追溯

查询链上记录的下载、修改、删除行为，全链路不可篡改、可追溯

暂无审计记录请输入资料 ID 或下载者地址查询链上行为。

可追溯内容

说明审计页能查询哪些链上行为。

下载记录谁下载了资料，何时下载
版本记录资料是否更新，哈希是否变化
删除记录谁执行删除，链上时间
  • 1按资料查查看某份资料的完整访问轨迹。
  • 2按地址查定位某个账户的下载行为。
  • 3完整追溯汇总上传、修改、下载和删除记录。
